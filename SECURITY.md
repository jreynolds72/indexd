# Security Policy

## Supported Versions

indexd is currently in beta. Security fixes are applied to the latest release on `main`.

## Reporting a Vulnerability

Please do not post security vulnerabilities in public issues.

Report security concerns by contacting the maintainer directly:
- GitHub: [@jreynolds72](https://github.com/jreynolds72)

Include as much detail as possible:
- Affected version/commit
- Reproduction steps or proof of concept
- Potential impact

You will receive acknowledgment as soon as possible, and we will coordinate on disclosure timing and remediation.
